## 🧠 Tech Stack
- React (Vite)
- Tailwind CSS
- React Router

## 🛠️ Installation

To run this project locally:
1. **Clone the repo:**
   ```bash
   
   git clone https://github.com/Ole12345678910/Portfolio2.git
   cd Portfolio2

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start development server:**

   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Visit `http://localhost:5173/` (or the port shown in your terminal)
